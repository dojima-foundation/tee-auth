# API Playground

Welcome to the TEE Auth API Playground! Here you can test all the available API endpoints interactively.

## Available Endpoints

### Activities
- **Approve Activity** - Approve a pending activity in the system

### Authentication
- **Create API Keys** - Create API keys for a user
- **Create Authenticators** - Create authenticators for secure access

### Transactions
- **Sign Transaction** - Sign a transaction using a secure enclave

### Organizations
- **Create Organization** - Create a new organization

### Users
- **Create User** - Create a new user in an organization

### Wallets
- **Create Wallet** - Create a new wallet for a user

### Private Keys
- **Create Private Key** - Create a new private key for a wallet

## Authentication

All API requests require authentication using the X-Stamp header. You'll need to provide your API key in the authentication field for each request.

## Interactive Examples

### Approve Activity

<RequestExample>
```json Request
{
  "type": "ACTIVITY_TYPE_APPROVE_ACTIVITY",
  "timestampMs": "1625097600000",
  "organizationId": "org_12345abcde",
  "parameters": {
    "activityId": "activity_67890fghij"
  }
}
```
</RequestExample>

<ResponseExample>
```json Response

```
</ResponseExample>

## Getting Started

1. Navigate to any endpoint page in the API Reference section
2. Look for the "Try it" button or interactive playground
3. Enter your X-Stamp authentication key
4. Fill in the required parameters
5. Click "Try it" to send the request

## Need Help?

If you encounter any issues with the API playground, please check:
- Your authentication credentials are correct
- All required parameters are filled in
- The API server is accessible at the configured base URL

For more detailed information about each endpoint, visit the individual endpoint pages in the API Reference section.
