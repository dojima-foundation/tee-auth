## List Wallets

This endpoint allows you to retrieve a list of all wallets owned by a specific user.

### Request

<ParamField path="organizationId" type="string" required>
  The ID of the organization
</ParamField>

<ParamField path="userId" type="string" required>
  The ID of the user
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of wallets to return (default: 50)
</ParamField>

<ParamField query="offset" type="number">
  Number of wallets to skip for pagination (default: 0)
</ParamField>

<ParamField query="walletType" type="string">
  Filter wallets by type (e.g., "ETH", "BTC", "GENERIC")
</ParamField>

<ParamField query="search" type="string">
  Search term to filter wallets by name or description
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="wallets" type="array">
  Array of wallet objects
</ResponseField>

<ResponseField name="wallets[].id" type="string">
  The ID of the wallet
</ResponseField>

<ResponseField name="wallets[].userId" type="string">
  The ID of the user who owns the wallet
</ResponseField>

<ResponseField name="wallets[].walletType" type="string">
  The type of the wallet
</ResponseField>

<ResponseField name="wallets[].name" type="string">
  The name of the wallet
</ResponseField>

<ResponseField name="wallets[].description" type="string">
  The description of the wallet
</ResponseField>

<ResponseField name="wallets[].publicKey" type="string">
  The public key of the wallet
</ResponseField>

<ResponseField name="wallets[].createdAt" type="string">
  The timestamp when the wallet was created
</ResponseField>

<ResponseField name="wallets[].privateKeyCount" type="number">
  Number of private keys associated with the wallet
</ResponseField>

<ResponseField name="total" type="number">
  Total number of wallets
</ResponseField>

<ResponseField name="limit" type="number">
  Number of wallets returned
</ResponseField>

<ResponseField name="offset" type="number">
  Number of wallets skipped
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/public/v1/organizations/org_12345abcde/users/user_67890fghij/wallets?limit=10&offset=0" \
  -H "Content-Type: application/json" \
  -H "X-Stamp: YOUR_STAMP"
```

```bash cURL with filter
curl -X GET "https://api.example.com/public/v1/organizations/org_12345abcde/users/user_67890fghij/wallets?walletType=ETH" \
  -H "Content-Type: application/json" \
  -H "X-Stamp: YOUR_STAMP"
```
</CodeGroup>
