## Create Private Key

This endpoint allows you to create a new private key for a wallet within a secure enclave.

### Request

<ParamField body="type" type="string" required>
  The activity type, must be `ACTIVITY_TYPE_CREATE_PRIVATE_KEY`
</ParamField>

<ParamField body="timestampMs" type="string" required>
  The timestamp in milliseconds when the request was created
</ParamField>

<ParamField body="organizationId" type="string" required>
  Your organization ID
</ParamField>

<ParamField body="parameters" type="object" required>
  The parameters object containing private key information
</ParamField>

<ParamField body="parameters.walletId" type="string" required>
  The ID of the wallet for which to create the private key
</ParamField>

<ParamField body="parameters.keyType" type="string" required>
  The type of private key to create (e.g., "ECDSA", "RSA", "ED25519")
</ParamField>

<ParamField body="parameters.keySize" type="number">
  The size of the key in bits (e.g., 256, 2048)
</ParamField>

<ParamField body="parameters.name" type="string">
  Optional name for the private key
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="privateKeyId" type="string">
  The ID of the created private key
</ResponseField>

<ResponseField name="walletId" type="string">
  The ID of the wallet that owns the private key
</ResponseField>

<ResponseField name="keyType" type="string">
  The type of the private key
</ResponseField>

<ResponseField name="publicKey" type="string">
  The public key corresponding to the private key
</ResponseField>
