## Delete Private Key

This endpoint allows you to delete a private key from a wallet.

### Request

<ParamField path="organizationId" type="string" required>
  The ID of the organization
</ParamField>

<ParamField path="userId" type="string" required>
  The ID of the user
</ParamField>

<ParamField path="walletId" type="string" required>
  The ID of the wallet
</ParamField>

<ParamField path="privateKeyId" type="string" required>
  The ID of the private key to delete
</ParamField>

<ParamField body="type" type="string" required>
  The activity type, must be `ACTIVITY_TYPE_DELETE_PRIVATE_KEY`
</ParamField>

<ParamField body="timestampMs" type="string" required>
  The timestamp in milliseconds when the request was created
</ParamField>

<ParamField body="parameters" type="object" required>
  The parameters object containing deletion information
</ParamField>

<ParamField body="parameters.confirmDeletion" type="boolean" required>
  Must be true to confirm the deletion of the private key
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="privateKeyId" type="string">
  The ID of the deleted private key
</ResponseField>

<ResponseField name="deletedAt" type="string">
  The timestamp when the private key was deleted
</ResponseField>

### Example Request

<CodeGroup>
```json Request
{
  "type": "ACTIVITY_TYPE_DELETE_PRIVATE_KEY",
  "timestampMs": "1625097600000",
  "parameters": {
    "confirmDeletion": true
  }
}
```

```bash cURL
curl -X DELETE "https://api.example.com/public/v1/organizations/org_12345abcde/users/user_67890fghij/wallets/wallet_12345abcde/private-keys/key_12345abcde" \
  -H "Content-Type: application/json" \
  -H "X-Stamp: YOUR_STAMP" \
  -d '{
    "type": "ACTIVITY_TYPE_DELETE_PRIVATE_KEY",
    "timestampMs": "1625097600000",
    "parameters": {
      "confirmDeletion": true
    }
  }'
```
</CodeGroup>
