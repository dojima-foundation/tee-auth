## Get Wallet

This endpoint allows you to retrieve details of a specific wallet.

### Request

<ParamField path="organizationId" type="string" required>
  The ID of the organization
</ParamField>

<ParamField path="userId" type="string" required>
  The ID of the user
</ParamField>

<ParamField path="walletId" type="string" required>
  The ID of the wallet to retrieve
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="wallet" type="object">
  The wallet object
</ResponseField>

<ResponseField name="wallet.id" type="string">
  The ID of the wallet
</ResponseField>

<ResponseField name="wallet.userId" type="string">
  The ID of the user who owns the wallet
</ResponseField>

<ResponseField name="wallet.walletType" type="string">
  The type of the wallet
</ResponseField>

<ResponseField name="wallet.name" type="string">
  The name of the wallet
</ResponseField>

<ResponseField name="wallet.description" type="string">
  The description of the wallet
</ResponseField>

<ResponseField name="wallet.publicKey" type="string">
  The public key of the wallet
</ResponseField>

<ResponseField name="wallet.createdAt" type="string">
  The timestamp when the wallet was created
</ResponseField>

<ResponseField name="wallet.updatedAt" type="string">
  The timestamp when the wallet was last updated
</ResponseField>

<ResponseField name="wallet.privateKeyCount" type="number">
  Number of private keys associated with the wallet
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/public/v1/organizations/org_12345abcde/users/user_67890fghij/wallets/wallet_12345abcde" \
  -H "Content-Type: application/json" \
  -H "X-Stamp: YOUR_STAMP"
```
</CodeGroup>
