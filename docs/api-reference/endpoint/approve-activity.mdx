## Approve Activity

This endpoint allows you to approve a pending activity in the system.

### Request

<ParamField body="type" type="string" required>
  The activity type, must be `ACTIVITY_TYPE_APPROVE_ACTIVITY`
</ParamField>

<ParamField body="timestampMs" type="string" required>
  The timestamp in milliseconds when the request was created
</ParamField>

<ParamField body="organizationId" type="string" required>
  Your organization ID
</ParamField>

<ParamField body="parameters" type="object" required>
  The parameters object containing activity-specific data
</ParamField>

<ParamField body="parameters.activityId" type="string" required>
  The ID of the activity to approve
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="activityId" type="string">
  The ID of the approved activity
</ResponseField>

<ResponseField name="status" type="string">
  The status of the activity after approval
</ResponseField>
