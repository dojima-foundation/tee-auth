## Create User

This endpoint allows you to create a new user within an organization.

### Request

<ParamField body="type" type="string" required>
  The activity type, must be `ACTIVITY_TYPE_CREATE_USER`
</ParamField>

<ParamField body="timestampMs" type="string" required>
  The timestamp in milliseconds when the request was created
</ParamField>

<ParamField body="organizationId" type="string" required>
  Your organization ID
</ParamField>

<ParamField body="parameters" type="object" required>
  The parameters object containing user information
</ParamField>

<ParamField body="parameters.email" type="string" required>
  The email address of the user
</ParamField>

<ParamField body="parameters.name" type="string" required>
  The full name of the user
</ParamField>

<ParamField body="parameters.role" type="string">
  The role of the user within the organization
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="userId" type="string">
  The ID of the created user
</ResponseField>

<ResponseField name="email" type="string">
  The email address of the user
</ResponseField>

<ResponseField name="name" type="string">
  The full name of the user
</ResponseField>

