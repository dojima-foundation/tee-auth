## List Private Keys

This endpoint allows you to retrieve a list of all private keys associated with a specific wallet.

### Request

<ParamField path="organizationId" type="string" required>
  The ID of the organization
</ParamField>

<ParamField path="userId" type="string" required>
  The ID of the user
</ParamField>

<ParamField path="walletId" type="string" required>
  The ID of the wallet
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of private keys to return (default: 50)
</ParamField>

<ParamField query="offset" type="number">
  Number of private keys to skip for pagination (default: 0)
</ParamField>

<ParamField query="keyType" type="string">
  Filter private keys by type (e.g., "ECDSA", "RSA", "ED25519")
</ParamField>

<ParamField query="search" type="string">
  Search term to filter private keys by name
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="privateKeys" type="array">
  Array of private key objects
</ResponseField>

<ResponseField name="privateKeys[].id" type="string">
  The ID of the private key
</ResponseField>

<ResponseField name="privateKeys[].walletId" type="string">
  The ID of the wallet that owns the private key
</ResponseField>

<ResponseField name="privateKeys[].keyType" type="string">
  The type of the private key
</ResponseField>

<ResponseField name="privateKeys[].name" type="string">
  The name of the private key
</ResponseField>

<ResponseField name="privateKeys[].publicKey" type="string">
  The public key corresponding to the private key
</ResponseField>

<ResponseField name="privateKeys[].createdAt" type="string">
  The timestamp when the private key was created
</ResponseField>

<ResponseField name="total" type="number">
  Total number of private keys
</ResponseField>

<ResponseField name="limit" type="number">
  Number of private keys returned
</ResponseField>

<ResponseField name="offset" type="number">
  Number of private keys skipped
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/public/v1/organizations/org_12345abcde/users/user_67890fghij/wallets/wallet_12345abcde/private-keys?limit=10&offset=0" \
  -H "Content-Type: application/json" \
  -H "X-Stamp: YOUR_STAMP"
```

```bash cURL with filter
curl -X GET "https://api.example.com/public/v1/organizations/org_12345abcde/users/user_67890fghij/wallets/wallet_12345abcde/private-keys?keyType=ECDSA" \
  -H "Content-Type: application/json" \
  -H "X-Stamp: YOUR_STAMP"
```
</CodeGroup>
