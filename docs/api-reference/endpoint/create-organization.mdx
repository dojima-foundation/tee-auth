## Create Organization

This endpoint allows you to create a new organization in the TEE Auth system.

### Request

<ParamField body="name" type="string" required>
  The name of the organization
</ParamField>

<ParamField body="initial_user_email" type="string" required>
  The email address of the initial admin user
</ParamField>

<ParamField body="initial_user_public_key" type="string">
  The public key of the initial admin user (optional)
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="data.organization.id" type="string">
  The ID of the created organization
</ResponseField>

<ResponseField name="data.organization.name" type="string">
  The name of the organization
</ResponseField>

<ResponseField name="data.user_id" type="string">
  The ID of the created admin user
</ResponseField>

<ResponseField name="data.status" type="string">
  The status of the creation operation
</ResponseField>

### Example Request

<CodeGroup>
```json Request
{
  "name": "Acme Corporation",
  "initial_user_email": "admin@acme.com",
  "initial_user_public_key": "acme-public-key"
}
```

```bash cURL
curl -X POST "http://localhost:8082/api/v1/organizations" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corporation",
    "initial_user_email": "admin@acme.com",
    "initial_user_public_key": "acme-public-key"
  }'
```
</CodeGroup>
