## Get Private Key

This endpoint allows you to retrieve details of a specific private key.

### Request

<ParamField path="organizationId" type="string" required>
  The ID of the organization
</ParamField>

<ParamField path="userId" type="string" required>
  The ID of the user
</ParamField>

<ParamField path="walletId" type="string" required>
  The ID of the wallet
</ParamField>

<ParamField path="privateKeyId" type="string" required>
  The ID of the private key to retrieve
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="privateKey" type="object">
  The private key object
</ResponseField>

<ResponseField name="privateKey.id" type="string">
  The ID of the private key
</ResponseField>

<ResponseField name="privateKey.walletId" type="string">
  The ID of the wallet that owns the private key
</ResponseField>

<ResponseField name="privateKey.keyType" type="string">
  The type of the private key
</ResponseField>

<ResponseField name="privateKey.name" type="string">
  The name of the private key
</ResponseField>

<ResponseField name="privateKey.publicKey" type="string">
  The public key corresponding to the private key
</ResponseField>

<ResponseField name="privateKey.createdAt" type="string">
  The timestamp when the private key was created
</ResponseField>

<ResponseField name="privateKey.updatedAt" type="string">
  The timestamp when the private key was last updated
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/public/v1/organizations/org_12345abcde/users/user_67890fghij/wallets/wallet_12345abcde/private-keys/key_12345abcde" \
  -H "Content-Type: application/json" \
  -H "X-Stamp: YOUR_STAMP"
```
</CodeGroup>
